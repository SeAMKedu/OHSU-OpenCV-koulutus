# OpenCV-koulutus
[![DOI](https://zenodo.org/badge/791715378.svg)](https://zenodo.org/doi/10.5281/zenodo.11066524)

Aloita matkasi OpenCV:n ihmeelliseen maailmaan tiedostosta [01-johdanto.md](01-johdanto.md).
